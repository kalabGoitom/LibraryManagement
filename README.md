# Library Management System

A **C++ project** that simulates a library system using **Object-Oriented Programming (OOP)** concepts like classes and objects.

---

## Features
- Add new books to the library
- Check book availability
- Search books by title or author
- Display all books
- Borrow and return books (optional if implemented)

---

## Technologies Used
- C++
- Classes and Objects
- Console input/output

---

## How to Run
1. Make sure you have a C++ compiler installed (like `g++`)  
2. Open terminal/command prompt in the project folder  
3. Compile the code:  
```bash
g++ *.cpp -o Library
